Welcome to the Postmast Emailer.

The purpose of this system is gather content from external sources and then email that content to a set of recipients.

The system has two parts: recipients and emails.

Recipients

The details of the person or entity. Recipients have the following fields:

  • First Name
  • Last Name
  • Email Address
  • Preferred Name

Recipients are aggregated into named groups called recipient groups. When creating an email, these groups are how you will define which recipients which recieve your email. Some important notes about recipient groups.

  • A recipient can be part of more than one recipient group.
  • A recipient can only be present in a recipient group once.
  • If an email is assigned to go to multiple recipient groups and a recipient is in more than one of those recipients groups, the recipient will only receive the email once.

Emails

This details what, when, and how an email will be sent to a set of recipients.
